# ⚡ ECE Circuit Calculator

A console-based circuit calculator built in C++ for electronics engineering students.

## Features

- 🔋 **Ohm's Law Calculator** — Find Voltage, Current or Resistance (V = IR)
- 🔌 **Series Resistance** — Calculate total series resistance
- 🔌 **Parallel Resistance** — Calculate total parallel resistance  
- ⚡ **Power Calculator** — Find Power, Voltage or Current (P = VI)
- 📡 **RC Circuit Calculator** — Time constant and charge time
- 💡 **LED Resistor Calculator** — Find limiting resistor for LED circuits

## How to Run

g++ circuit_calculator.cpp -o circuit
./circuit

## Concepts Used

- Object Oriented Programming (OOP)
- Classes & Objects
- Functions & Loops
- C++ Math Library
